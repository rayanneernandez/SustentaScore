**Sistema de Monitoramento de Sustentabilidade em Contratos Públicos**

## Sobre o Projeto

O **SustentaScore** é uma solução digital voltada para órgãos públicos que precisam monitorar o cumprimento de critérios de sustentabilidade nos contratos com fornecedores. O sistema centraliza o cadastro de fornecedores, contratos e indicadores ESG, calcula automaticamente a pontuação de desempenho (score) e oferece dashboards analíticos para tomada de decisão.

---

## Funcionalidades

| Módulo | Descrição |
|---|---|
| 📊 Painel Gerencial | Dashboard com visão consolidada de scores, ocorrências e evolução histórica |
| 🏢 Cadastro | Gestão de fornecedores e contratos com todos os dados administrativos |
| 🌱 Indicadores | Configuração de indicadores ESG por eixo temático e tipo de contrato |
| ⚠️ Ocorrências | Registro de não conformidades com dedução automática de pontos |
| 🏆 Cálculo do Score | Motor de pontuação com faixas Verde, Cinza e Preto |
| 💳 Medição e Pagamento | Apuração mensal com liberação ou bloqueio de pagamento por desempenho |

---

## Faixas de Score

| Faixa | Pontuação | Significado |
|---|---|---|
| 🟢 Verde | 450 – 500 | Desempenho excelente — pagamento integral |
| ⚫ Cinza | 350 – 449 | Desempenho regular — monitoramento reforçado |
| 🔴 Preto | Abaixo de 350 | Desempenho crítico — possível bloqueio de pagamento |

## Tecnologias

- **React 18 + TypeScript** — SPA com tipagem estrita
- **Vite** — bundler e dev server ultrarrápido
- **React Router DOM v6** — roteamento entre as 6 páginas
- **Recharts** — gráficos de linha, barra horizontal e donut
- **Lucide React** — ícones consistentes
- **CSS Custom Properties** — design system próprio sem framework
